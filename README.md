# gouillotine

Telegram bot that "cuts off" 13:37 time at 13:38.

## Env vars

- `TELEGRAM_BOT_TOKEN`: The bot API token
- `TELEGRAM_GROUP_ID`: The group ID to target

## Logging

Set `RUST_LOG=guillotine=debug` (or `info` or `trace`).
